Goal

Validate that the rebuilt docs:plans convention works end-to-end: a plan file using the new schema can be created, listed in three pretty-print tiers, moved through planned/ → ongoing/ → finished/ with started_at set and ship_commit on ship, and reviewed by the new plan-review skill which writes a structured ## Review block.

This smoke-test plan exists to dogfood the result — and is closed by shipping itself through the system it validates.
